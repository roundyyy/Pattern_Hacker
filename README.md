# Pattern Hacker

A mobile-friendly memory game with a hacker aesthetic. Watch the lines drawn between nodes, memorize the pattern, then trace it back.

**[Play now](https://roundyyy.github.io/Pattern_Hacker/)**

## Gameplay

- Watch lines light up between nodes on a grid, then repeat the pattern by dragging through them
- Each level adds one more line to memorize
- The grid starts at 2x2 and expands up to 6x6 as you progress
- You have limited lives, a countdown timer, and one "Show Again" per level
- Build combos by clearing consecutive levels without mistakes
- Earn Decrypt charges from perfect clears to auto-fill the first few nodes
- Extra lives are awarded every 3 levels

## Difficulty modes

- **Normal** - the default experience with leaderboard support
- **Easy** - more lives, slower pattern display, more time, shorter patterns, and patterns that favor straight horizontal/vertical lines. Leaderboard submission is disabled in easy mode.
